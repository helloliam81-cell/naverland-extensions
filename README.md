# 네이버 부동산 단지 트레킹

네이버 부동산에서 여러 단지의 매매/전세 최저호가를 한눈에 조회하는 Tampermonkey 스크립트입니다.

---

## 주요 기능

- 관심 단지를 트레킹 목록에 추가
- 매매/전세 최저호가 일괄 조회
- 층수 필터 설정 (저층 포함 여부, N층 이하 제외)
- 전용면적별 최저가 조회
- 트레킹 목록 영구 저장 (브라우저 재시작 후에도 유지)

---

## 설치 방법

### 1단계: Tampermonkey 설치

사용하는 브라우저에 맞게 Tampermonkey 확장 프로그램을 설치합니다.

| 브라우저 | 설치 링크 |
|---|---|
| Chrome | [Chrome 웹 스토어](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo) |
| Edge | [Microsoft Store](https://microsoftedge.microsoft.com/addons/detail/tampermonkey/iikmkjmpaadaobahmlepeloendndfphd) |
| Firefox | [Firefox 부가 기능](https://addons.mozilla.org/ko/firefox/addon/tampermonkey/) |

설치 후 브라우저 우측 상단에 Tampermonkey 아이콘이 나타나면 준비 완료입니다.

---

### 2단계: 스크립트 설치

아래 버튼을 클릭하면 Tampermonkey가 자동으로 설치 화면을 띄웁니다.

**[▶ 스크립트 설치하기](https://github.com/helloliam81-cell/naverland-extensions/raw/main/naverland-tracking.user.js)**

> 링크를 클릭했는데 아무 반응이 없다면 Tampermonkey가 제대로 설치되지 않은 것입니다. 1단계를 다시 확인해 주세요.

설치 화면이 열리면 **"설치"** 버튼을 클릭합니다.

![설치 화면 예시](https://imgur.com/a/5zzEzBL)

---

### 3단계: 사용 확인

1. [네이버 부동산](https://new.land.naver.com)에 접속합니다.
2. 화면 우측 상단에 **📍 트레킹 (0)** 초록 버튼이 보이면 설치 완료입니다.

## 사용 방법

### 단지 추가

1. 네이버 부동산에서 관심 단지 페이지로 이동합니다.
2. 단지명 옆에 나타나는 **📌 아이콘**을 클릭합니다.
3. 아이콘이 **📍**로 바뀌면 트레킹 목록에 추가된 것입니다.

### 호가 조회

1. 우측 상단 **📍 트레킹** 버튼을 클릭하여 목록을 엽니다.
2. **전체 실행** 버튼을 클릭하면 모든 단지의 최저호가를 조회합니다.
3. 개별 단지만 조회하려면 해당 행의 **실행** 버튼을 클릭합니다.

### 단지 페이지 이동

목록에서 **단지명을 클릭**하면 해당 단지의 상세 페이지로 이동합니다.

> 전체 실행 중 단지명을 클릭해 페이지가 이동되더라도, 트레킹 창은 자동으로 다시 열리고 남은 단지의 조회가 이어서 진행됩니다.

### 층수 필터

목록 상단의 **층수 필터** 영역에서 설정합니다.

| 항목 | 설명 |
|---|---|
| 저층 포함 / 저층 미포함 | 매물 정보에 "저층"으로 표기된 매물 포함 여부 |
| N층 이하 제외 | 설정한 층수 이하 매물을 결과에서 제외 |

> 기본값: 매매는 저층 미포함·3층 이하 제외 / 전세는 저층 포함·제외 없음
> 필터 변경 후에는 **실행** 버튼을 다시 눌러야 반영됩니다.

### 전용면적 선택

조회 완료 후 전용면적 드롭다운에서 특정 면적만 선택하면 해당 면적의 최저가만 표시됩니다.

## 자주 묻는 질문

**Q. 버튼이 보이지 않아요.**
A. 네이버 부동산 사이트(`new.land.naver.com`)에서만 동작합니다. 다른 주소에서는 표시되지 않습니다.

**Q. "오류"라고 표시돼요.**
A. 네이버 부동산 서버 응답이 일시적으로 실패한 경우입니다. 잠시 후 다시 실행해 보세요.

**Q. 트레킹 목록이 사라졌어요.**
A. 목록은 브라우저의 localStorage에 저장됩니다. 브라우저 데이터를 초기화하면 목록도 삭제됩니다.

**Q. 매물없음이라고 떠요.**
A. 현재 설정된 층수 필터 조건에 맞는 매물이 없는 경우입니다. 층수 필터를 조정해 보세요.
